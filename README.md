# Faast PWA
Complete source code of the Faast progressive web application.

## Running locally
```
git clone https://github.com/JeetHathi/faast-pwa.git
cd faast-pwa
npm install
npm start
```

## Or you can use yarn
```
git clone https://github.com/JeetHathi/faast-pwa.git
cd faast-pwa
yarn install
yarn run start
```
